# momoa 모모아


## 프로젝트 소개
모모아는 수입과 지출 관리, 분석을 위한 가계부 웹입니다. 또한 유저 간 공유 기능을 통해 가계부를 공동으로 관리할 수 있도록 했습니다.


### 개발 기간
2023.02.11 ~ 2023.02.25 (2주간)

### 개발 인원
김세현, 박의현, 김미정, 장현수(4명)
